Estrutura básica do HTML

  • Significa Hyper Text Markup Language
  • É uma linguagem de marcação padrão para criar páginas da Web
  • Ele descreve a estrutura de uma página da Web
  • Consiste em uma série de elementos
  • Os elementos HTML informam ao navegador como exibir o conteúdo

HTML é usada para a publicação de conteúdo (texto, imagens, vídeos, áudio etc.) na web, baseado no conceito de hipertexto, que são conjuntos de elementos ligados por conexões.

Modelo de estrutura.

!DOCTYPE HTML - Instrução que indica ao navegador qual a versão de marcação o código foi escrito.

html lang=”pt-br” - Elemento HTML. O atributo “lang” é necessário para que saibam qual é a linguagem principal do documento.

heade - Local onde fica a “parte inteligente” do web site. No HEAD, ficam as informações sobre a página e sobre o conteúdo.

meta charset=”UTF-8” - Tabela que comporta um milhão de caracteres e a maioria dos browsers a suporta plenamente, ela pode acessar qualquer conteúdo disponível na web.

title - Título que aparece no navegador ou na guia da página

body - Corpo do documento e é um recipiente para todos os conteúdos visíveis, como cabeçalhos, parágrafos, imagens, hiperlinks, tabelas, listas, etc.

O que são as TAGS

(Voltar)

As tags são formadas por uma estrutura própria, iniciam com o sinal “menor que” (<) em seguida vem o nome daquele elemento e por fim, o sinal “maior que iniciando com uma barra” (/ >).

Exemplo:< tagnome > Conteúdo < /tagnome >.

Tags semânticas - São tags que possuem um significado, que dão sentido a informação de texto ao navegador, ajudar a entender o código.

Atributos das tags

são palavras especiais usadas dentro da tag de abertura para controlar o comportamento do elemento.

Há dois tipos de atributos no HTML: os globais que são aceitos por todas as tags, como por exemplo: class, id, e os específicos que somente algumas tags possuem, como src, href, etc…

Exemplo:
id=” ” -É utilizado para identificar destino de âncoras
src=” ” - Link ou diretório da mídia" utilizado para indicar para a tag qual arquivo ou mídia utilizar. Recebe valores como links ou o nome de um arquivo já presente no projeto.
target=”blank” - Esse atributo abre o link do documento em uma nova janela ou aba.
href=”Url” - Para a tag , o atributo href especifica a URL da página para a qual o link vai.

Fundamentos do texto em HTML

(Voltar)

Um dos principais objetivos do HTML é dar estrutura de texto e significado, também conhecido como semântica. (se refere ao significado de um trecho de código).

Um texto estruturado é composto por títulos e parágrafos.

Existem seis elementos de título h1, h2, h3, h4, h5 e h6. Cada elemento representa um nível diferente de conteúdo no documento; < h1 > representa o título principal, < h2 > representa subtítulos, < h3 > representa sub-subtítulos, e assim por diante e cada parágrafo deve ser envolvido em um elemento < p >.

Listas

(Voltar)

Para exibir uma listagem em uma páginas. O HTML tem algumas tags definidas.

Ordenada

Listas ordenadas são listas em que a ordem dos itens é importante.

A estrutura de marcação das listas ordenadas deve conter itens em um elemento < ol> e envolvido em uma < li> (list item)

Não ordenada

As listas não ordenadas são usadas para marcar itens as quais a ordem não importa. Toda lista desordenada começa com um < ul> isso envolve todos os itens e cada item da lista é envolvido em uma < li> (list item).

Links no HTML

(Voltar)

Os links HTML são definidos com a tag < a> é a forma abreviada de "âncora".

A tag define um hiperlink. O href atributo especifica a URL da página para a qual o link vai. Exemplo: < a href="https://www.endereço.com"> Este é um link < /a>.

Podemos adicionar o atributo target=" " para especificar onde que essa página irá carregar. Por padrão a página irá abrir na mesma aba da página que tem o link, mas se quisermos que a página abra em outra aba podemos colocar o valor _blank dentro desse atributo: < a href="https://www.endereço.com.br" target="_blank">< /a>.

As âncoras que começam com [#], você adiciona um href=# onde a hashtag [#] índica que o link está na mesma página e trata-se de um id. O ID não precisa de hashtag, mas de palavra-chave.

Imagens

As imagens em HTML são definidas com a tag. < img>.

O arquivo de origem ( src= atributo especifica o caminho para a imagem a ser exibida), ( alt= especifica um texto alternativo para uma imagem, se a imagem por algum motivo não puder ser exibida).

Exemplo: < img src="img_gato.jpg" alt="Foto de um animal gato">.
